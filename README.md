# Polytone Patch Merger

Web app to combine two patches for [Reason's Polytone synth](https://www.reasonstudios.com/devices/polytone) into a single patch.

## Dependencies

* node >=20
* php >= 7

## Getting Started

* `npm install`
* `npm run index`
* `npm start`
